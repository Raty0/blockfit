# 🎮 BlockFit 玩家更新笔记 (Update Notes)

我们将严格遵守版本记录规则：**大版本 (X.0.0) -> 功能调整 (0.X.0) -> 小修复 (0.0.X)**。

---

## [v4.1.0] - 2026-04-08
### 🚀 关卡平衡与难度升级 (Balancing)
- **挑战性大幅提升**：我们优化了关卡生成引擎，让难度曲線不再缓慢增长。现在前几百关的网格将更快变大，方块组合也会更加复杂。告别简单模式，迎接真正的逻辑挑战！

## [v4.0.2] - 2026-04-08
### 🐛 体验修复 (Small Fix)
- **真正的音频修复**：抛弃了传统的音频播放模式，全面启用了专业的 `Web Audio API`。现在无论是放着等多久，只要你一动手，音效绝对100%饱满、清脆，永远不会再变小声了。

## [v4.0.1] - 2026-04-08
### 🐛 体验修复 (Small Fix)
- **终极音效修复**：重写了整个底层音效引擎。无论你等待多久，音效都不会再变小了，因为每一次放置都会为你全新生成一次100%音量的声音。

## [v4.0.0] - 2026-04-08
### 🚀 玩法大升级 (Big Change)
- **一键回退方块**：不小心放错了？现在只需要**轻点一下**已放置的方块，它就会立即消失，恢复到空白状态。放轻松，不用再一步步按撤销了！
- **全新彩虹色彩系统**：每种不同的数字都拥有 100% 独一无二的颜色，让你一眼就能区分不同的方块区域，再也不会搞混了。
- **放置预览功能**：拖拽方块时，屏幕上会先出现一个**半透明的预览**，让你精确确认位置后再松手放置。预览方块使用同色系的淡色填充 + 深色边框，拖动手感更有掌控力。

## [v3.1.1] - 2026-04-08
### 🐛 体验修复 (Small Fix)
- **音效音量修复**：解决了在某些情况下游戏音效会变得非常小的问题。现在系统会强制在每次播放时恢复 100% 音量，防止被手机系统自动调低。

## [v3.1.0] - 2026-04-08
### ✨ 体验优化 (Small Change)
- **新增加载转场动画**：为了让你在进入关卡时有更完美的视觉体验，并确保所有音效、资源都已就绪，我们加入了一个极简美观的加载页面。

## [v3.0.2] - 2026-04-08
### 🐛 体验修复 (Small Fix)
- **音效稳定性修复**：彻底解决了放置方块时音效播放不稳定的问题。现在每次落子都会有清脆的反馈。

## [v3.0.1] - 2026-04-08
### 🐛 细节优化 (Small Fix)
- **修复导航失效**：解决了一个在游戏内点击设置按钮偶尔导致界面卡住的小问题。

## [v3.0.0] - 2026-04-08
### 🚀 史诗级内容更新 (Big Change)
- **挑战无限：3000 关卡**：关卡总数从 300 关疯狂扩充至 **3000 关**！
- **脑力竞技：全员同步**：引入了全新的关卡生成算法。现在，你和你的朋友、全球玩家在同一关看到的解法都是完全一致的，快来切磋吧！
- **瞬移功能：关卡搜索**：在选关界面新增了“放大镜”图标，输入数字直接跳转，再多关卡也不怕！
- **贴心定位：自动滚动**：进入选关列表时，系统会自动定位到你目前正在挑战的那一关，告别手动查找。
- **视觉瘦身**：去除了首页多余的格子信息，界面更加简洁纯粹。

## [v2.1.0] - 2026-04-08
### ✨ 功能优化 (Small Change)
- **品牌升级**：正式更名为 **BlockFit**！更响亮，更好记。
- **顺溜滑动**：彻底修复了选关界面的滑动锁定问题。现在只需随手一挥，关卡列表如丝般顺滑。
- **美化侧边栏**：侧边滚动条新增了渐变紫色彩，并完美贴合屏幕边缘。

## [v2.0.0] - 2026-04-08
### 🚀 界面全面焕新 (Big Change)
- **全新首页（Home）**：进游戏第一眼就是专属首页，一键 Resume，操作直达。
- **旗舰级设置页**：设置界面不再是小弹窗。我们专门建立了一个全屏的设置页面，并预留了未来的自定义功能。
- **悬浮导航栏**：屏幕下方加入了悬浮式的毛玻璃效果菜单，在 Home 和 Settings 之间自由穿梭。
- **视觉平衡**：重新调整了游戏棋盘的位置，让你玩游戏时视线始终保持在屏幕中央。

## [v1.1.0] - 2026-04-07
### ✨ 系统微调 (Small Change)

- **流程优化**：改进了关卡加载速度，玩游戏更流畅。

## [v1.0.0] - 2026-04-07
### 🎉 最初的起点 (Big Change)
- **Shikaku 引擎**：首发 300 关解密关卡。
- **核心玩法**：经典的拉拽填方块、数字逻辑匹配引擎正式上线。
- **自动存档**：支持关卡进度自动保存，随时玩耍。
